Static Web Pages : Process Overview

  • Client sends a request to the server.
  • Server receives the request and sends it to the Django framework.
  • Django framework looks for the requested page in the urls.py file.
  • If the page is found, Django framework sends the request to the views.py file.
  • views.py file processes the request and sends it back to the Django framework.
  • Django framework sends the response to the server.
  • Server sends the response to the client.
